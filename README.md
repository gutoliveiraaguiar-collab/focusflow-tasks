Um aplicativo de lista de tarefas (To-Do List) moderno, responsivo e com persistência local. Desenvolvido como projeto de estudo para praticar HTML, CSS, Bootstrap 5 e JavaScript.

## 🚀 Funcionalidades

- ✅ **Adicionar tarefas** (botão ou tecla Enter)
- ✅ **Marcar como concluída** (clique no texto ou ícone)
- ✅ **Excluir tarefa individual** (ícone de lixeira)
- ✅ **Limpar todas as concluídas** (botão dedicado)
- ✅ **Contador de tarefas pendentes** (atualiza em tempo real)
- ✅ **Persistência com localStorage** (tarefas salvas no navegador)
- ✅ **Animações suaves** (fadeIn, fadeOut, hover)
- ✅ **Totalmente responsivo** (mobile, tablet, desktop)
- ✅ **Tarefas de exemplo** na primeira visita

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
| :--- | :--- |
| **HTML5** | Estrutura semântica da página |
| **CSS3** | Estilização customizada, gradientes, animações |
| **Bootstrap 5.3** | Grid responsivo, componentes (Cards, Navbar, Buttons) |
| **Bootstrap Icons** | Ícones vetoriais |
| **JavaScript (ES6+)** | Lógica do CRUD, manipulação do DOM, localStorage |
